# ocr4u
OCR: optical character recognition

ocr4u uses OCRmyPDF to add OCR to a PDF or redo OCR for a PDF with existing OCR. 
Currently, ocr4u only accepts PDFs, but it will accept images in the future.

##Commands
- `ocr4u add_ocr -f <filename> -o <outputPath>`: add OCR to a PDF
- `ocr4u redo_ocr -f <filename> -o <outputPath>`: redo existing OCR in a PDF
